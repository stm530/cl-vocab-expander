# 人工言語語彙拡充器 (cl-vocab-expander)

日本語WordNetを刺激（プロンプト）として、人工言語の語彙を拡充するWebアプリケーションです。
作者が個々の語に対してその場で意味範囲の判断を下す、受動的・反復的な語彙獲得プロセスを提供します。

公開サイト: **<https://stm530.github.io/cl-vocab-expander/>**

本アプリはサーバー側にデータを持たない静的Webアプリであり、作業中の辞書データは
お使いのブラウザ内にのみ保存されます。アカウント登録やログインは不要です。

## 対象とする辞書

[ZpDIC Online](https://zpdic.ziphil.com/) に登録されている辞書であれば、
言語を問わず利用できます。本アプリが読み書きするのは以下の4項目のみです。

- 単語（見出し語の綴り）
- 品詞（複数選択可）
- 意味（訳語）
- 関連語

それ以外のパラメータ（発音記号・セクションなど）は、アップロードした元データの値を
そのまま保持し、本アプリから編集することはありません。

## 使い方

### 1. 辞書データを取り込む

1. ZpDIC Online から対象辞書を JSON (OTM-JSON) 形式でエクスポートする
2. 「辞書」タブでファイルをアップロードする
3. WordNet データベースの読み込みが完了するのを待つ（初回のみ数十MBのダウンロードがあります）

### 2. 語彙を拡充する

- **出題タブ**: WordNetのsynset（意味概念）がランダムに提示されます。綴り・品詞・訳語を入力して提出してください。
- **意味追加タブ**: 既存辞書にある語に新しい意味（多義）を追加します。ランダム出題を経由せず、見出し語を検索して着手できます。
- **レビュータブ**: 追加した内容を一覧・承認し、更新済みZPDCファイルをダウンロードします。
- **設定タブ**: 出題オプションの調整、作業中データのエクスポート/インポートを行います。

### 3. 拡張データをZpDICに戻す

1. 「レビュー」タブで収録した内容を承認する
2. 「承認済み反映」ボタンで更新済みZPDCファイルをダウンロード
3. ZpDIC Online のWebインポート機能で手動アップロードする

本アプリはZpDIC OnlineのAPIに直接接続しません。新語の反映はファイル単位の
手動アップロードにより行ってください。

## 作業データの保存とバックアップ

作業中の辞書データはブラウザ内のIndexedDBに自動保存されますが、ブラウザの
「閲覧データ消去」操作やプライベートブラウジングモードでは失われるリスクが
あります。消失を防ぐため、こまめに **設定タブからJSONファイルをエクスポート** し、
バックアップを手元に保存することを推奨します。保存したJSONファイルは同じ画面から
読み込むことで作業を再開できます。
